About The Project

The aim of this project was to study UFO sightings from the year of 2016, and display our findings on a delightfully retro webpage! We found this dataset on Kaggle and it includes over 5,000 sightings. We wanted to focus on the location of sightings, their descriptions, and other trends found in the data.


First we cleaned the data in Python using the Pandas library. The date strings were inconsistent in format and we wanted them to be in datetime format. Using the tips we found from StackOverflow, we converted the dates.


Then, we imported the data into a Postgres SQL database. Using SQLAlchemy we accessed this database to create a Flaskapp. On this app we hosted an API containing our Jsonified data.


Using our API, we called the data into our Javascript file through the D3 library. We used Leaflet to display the sightings on an interactive map, and the Chart.js library to create a dashboard detailing popular trends in our data.


Finally, we created a webpage to display our results using css styling aiming to capture the retro-charm of websites from the 90s!
